# Vue使用fliters

# 局部组件使用

<script>
export default {
	filters: {
		toFixed(value) {
			if(value) {
				return value.toFixed(2)
			}
		}
	}
}
</script>

# 全局使用

# 1、在 src 中创建 fliters文件夹,创建index.js

export function toFixed(value) {
	if(value) {
		return value.toFixed(2)
	}
}

# 2、在main.js使用注册

import * as filters from './filters'

Object.keys(filters).forEach(key => {
	Vue.filter(key,filters[key])
})

# Vue中使用axios

Vue中使用axios有两种办法：

# 1. 结合 vue-axios使用

vue-axios 是照着vue插件的方式去写的。那么结合vue-axios，就可以使用vue.use()方法了。

  1. 首先在主入口文件main.js引入
import axios from 'axios';
import VueAxios from 'vue-axios';

Vue.use(VueAxios,axios);
  1. 然后就可以在组件中的methods方法里面使用了：
getData() {
	this.axios.get().......
}

# 2. axios改为Vue的原型属性

  1. 首先在主入口文件main.js中引入，之后挂在Vue的原型链上;
import axios from 'axios';
Vue.prototype.$axios = axios;
  1. 在组件中使用
this.$axios.get().....[]()
